# Dragon Paving Website

## How to use (GitHub Pages)
1. Upload all files in this folder to a GitHub repository
2. Add your homepage image and name it: `home.png`
3. Go to **Settings → Pages**
4. Set Source to **main / root**
5. Visit the provided GitHub Pages URL

## Files
- index.html (homepage – exact image mockup)
- home.png (YOU must upload this image)

If the page is blank, it means `home.png` is missing.
